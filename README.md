# FiftyTwo Pickup

FiftyTwo Pickup is a web application that lets you practice memorizing a shuffled deck of 52 playing cards, a common discipline between memory athletes. The app is built with **Python**, **Flask**, and **SQLite**.

## Features

- Simple web interface to practice card memorization.
- Tracks each user’s attempts and performance.
- Stores data securely in a SQLite database located in the `instance/` folder.
